# IEEE SB REC Website

Official website for the IEEE Student Branch at REC Sonbhadra.

---

## 🚀 Features

* Modern Responsive UI
* Event Showcase
* Team Section
* IEEE Chapters Overview
* Registration Portal
* Admin Dashboard

---

## 🛠 Tech Stack

* React.js
* Vite
* Tailwind CSS
* Firebase
